How many donuts to bake per hour/day?

This simulator is created to help each donut shop predict the amount of donuts they should be producing each day. The simulator has been calculated based on the following information:

  • The hours of operation of each shop (in this case we use 11 hours as the usual shop hours are 7am-6pm).
  • A random sample of customers per hour (this is a computer-based calculation).
  • The average number of donuts purchased per customer (this has been pre-calculated based on the historical information).

Please select your shop location below to see the results. If you want to see all locations, please select "All Locations".
